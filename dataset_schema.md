# F1 Master Dataset Schema

This document outlines the columns generated by the upgraded data collection and feature engineering pipelines, saved to `master_f1_dataset.csv`.

## 1. Identifiers & Context (Categorical/Ordinal)
| Feature | Type | Description |
| :--- | :--- | :--- |
| `Year` | Int | Championship Year (e.g., 2024) |
| `EventName` | String | Name of the Grand Prix |
| `Country` | String | Host Country |
| `Circuit` | String | Name/Location of the Circuit |
| `RoundNumber` | Int | Race number in the calendar year |
| `SessionType` | String | 'Race', 'Qualifying', 'Sprint', etc. |
| `Driver` | String | 3-Letter Driver Abbreviation (e.g., 'VER') |
| `Team` | String | Constructor Name (e.g., 'Red Bull Racing') |
| `Compound` | String | Tyre Compound used (e.g., 'SOFT', 'MEDIUM', 'HARD') |
| `LapNumber` | Int | The specific lap number within the session |

## 2. Race & Track Status (Boolean/Categorical)
| Feature | Type | Description |
| :--- | :--- | :--- |
| `IsSafetyCar` | Boolean | True if the Safety Car was deployed during the lap. |
| `IsVirtualSafetyCar` | Boolean | True if the Virtual Safety Car was deployed during the lap. |
| `IsStreetCircuit` | Boolean | True if the track is a temporary street circuit. |
| `IsHighSpeed` | Boolean | True if the track is classified as a high-speed circuit. |
| `IsHighDownforce` | Boolean | True if the track requires high downforce setups. |
| `IsPermanentCircuit` | Boolean | True if it's a dedicated racing facility. |

## 3. Weather Conditions (Numeric)
| Feature | Type | Description |
| :--- | :--- | :--- |
| `AirTemp` | Float | Ambient air temperature (Celsius) |
| `TrackTemp` | Float | Track surface temperature (Celsius) |
| `Humidity` | Float | Relative humidity percentage |
| `WindSpeed` | Float | Wind speed (m/s) |
| `Rainfall` | Boolean | True if it is currently raining |

## 4. Advanced Race Features (Numeric)
| Feature | Type | Description |
| :--- | :--- | :--- |
| `FuelEffect` | Float | The estimated lap time advantage (seconds) gained due to fuel burn up to this point in the race. |
| `TrackEvolution` | Float | The cumulative minimum lap time achieved across all drivers in the session so far, serving as a proxy for how "rubbed in" and fast the track has become. |

## 5. Tire & Race Engineering (Numeric)
| Feature | Type | Description |
| :--- | :--- | :--- |
| `TyreLife` | Int | The age of the current tyre set (in laps). |
| `TireDegradationIndex` | Float | The time difference between the current lap and the driver's best lap on this specific set of tyres. |
| `PositionChange` | Int | Places gained/lost relative to the driver's starting grid position. |
| `StartingPosition` | Int | The driver's grid slot at the start of the race/sprint. |

## 6. Rolling Pace & Consistency (Numeric)
| Feature | Type | Description |
| :--- | :--- | :--- |
| `PrevLapTime` | Float | The driver's exact lap time from the previous lap (Seconds). |
| `AvgLast[3/5/10]Laps` | Float | Rolling average lap time over the last X laps. |
| `DriverConsistency[3/5]Laps` | Float | Rolling standard deviation of lap times over the last X laps (measures consistency). |

## 7. Historical Telemetry & Sectors (Numeric)
*Note: Telemetry features are prefixed with `Prev` to prevent data leakage, ensuring the model only trains on telemetry data available BEFORE the current lap finishes.*

| Feature | Type | Description |
| :--- | :--- | :--- |
| `PrevSector[1/2/3]TimeSeconds` | Float | The driver's sector splits from the previous lap. |
| `PrevAvgSpeed / MaxSpeed / MinSpeed` | Float | Speed telemetry (km/h) from the previous lap. |
| `PrevAvgThrottle / MaxThrottle` | Float | Throttle pedal engagement percentage. |
| `PrevBrakePercentage` | Float | Percentage of the lap spent braking. |
| `PrevDRSPercentage` | Float | Percentage of the lap with the DRS flap open. |
| `PrevAvgRPM / MaxRPM` | Float | Engine RPM telemetry. |
| `PrevCorneringSpeed` | Float | Minimum speed through corners (10th percentile speed). |
| `PrevAvgGear` | Float | Average gear used throughout the previous lap. |

## 8. Target Variables
| Feature | Type | Description |
| :--- | :--- | :--- |
| `LapTimeSeconds` | Float | **(Primary Target)** The actual total lap time in seconds. |
| `Position` | Int | **(Secondary Target)** The final race classification/position of the driver. |
